## How to Contribute

### Reporting Issues

If you find a bug or have a feature request:
1. Check if the issue already exists in [GitHub Issues](https://github.com/fatdinhero/fog-brave-site/issues)
2. If not, create a new issue with:
   - Clear title and description
   - Steps to reproduce (for bugs)
   - Expected vs actual behavior
   - Screenshots (if applicable)

### Code Contributions

1. **Fork the repository**
2. **Create a feature branch**
   ```bash
   git checkout -b feature/your-feature-name
   ```

3. **Make your changes**
   - Follow existing code style
   - Keep changes focused and atomic
   - Test your changes thoroughly

4. **Commit your changes**
   ```bash
   git commit -m "Add: Brief description of your changes"
   ```
   
   Use conventional commit prefixes:
   - `Add:` for new features
   - `Fix:` for bug fixes
   - `Update:` for improvements
   - `Remove:` for deletions
   - `Docs:` for documentation

5. **Push to your fork**
   ```bash
   git push origin feature/your-feature-name
   ```

6. **Create a Pull Request**
   - Provide a clear description of changes
   - Reference any related issues
   - Include screenshots for UI changes

## Code Style Guidelines

### HTML
- Use semantic HTML5 elements
- Maintain proper indentation (4 spaces)
- Add comments for complex sections
- Keep accessibility in mind

### CSS
- Use CSS custom properties (variables) for consistency
- Follow BEM naming convention where appropriate
- Maintain responsive design principles
- Test on multiple devices

### JavaScript
- Use modern ES6+ syntax
- Keep functions small and focused
- Add comments for complex logic
- Handle errors gracefully
- Test Web3 functionality with multiple wallets

## Testing

Before submitting:
1. Test on major browsers (Chrome, Firefox, Safari, Brave)
2. Test responsive design on mobile devices
3. Test Web3 wallet connections (MetaMask, Phantom, Brave Wallet)
4. Verify all links work
5. Check console for errors

## Adding New Blockchain Support

To add a new blockchain:
1. Add wallet card in the wallets section
2. Choose appropriate symbol and color
3. Provide valid wallet address
4. Update README.md with new chain info
5. Test copy functionality
